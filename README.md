
## ABOUT:
Automatically brute force all services running on a target

* Open ports
* Usernames
* Passwords

## INSTALL:
```
./install.sh
```

## USAGE:
```
bruter target <port>
```

## DOCKER:
```
docker build -t bruter .
docker run -it bruter target <port>
```
